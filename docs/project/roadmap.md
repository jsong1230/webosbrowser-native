## Milestone 1: MVP Core — 기본 브라우징

### 목표
webOS 프로젝터에서 리모컨으로 웹 페이지를 로드하고 기본적인 탐색이 가능한 브라우저 핵심 기능 구현

### 포함 기능
- **F-01**: 프로젝트 초기 설정
- **F-02**: 웹뷰 통합
- **F-03**: URL 입력 UI
- **F-04**: 페이지 탐색 컨트롤
- **F-05**: 로딩 인디케이터

### 완료 기준
- 앱 실행 시 브라우저 UI 표시
- 리모컨으로 URL 입력 가능
- URL 입력 후 웹 페이지 렌더링
- 페이지 로딩 중 프로그레스바 표시
- 뒤로/앞으로/새로고침/홈 버튼 동작
- YouTube, Google 등 주요 사이트 정상 렌더링

### 주요 산출물
- webOS Enact 프로젝트 구조
- WebView 컴포넌트 (웹 페이지 렌더링 엔진)
- URLBar 컴포넌트 (리모컨 가상 키보드)
- NavigationBar 컴포넌트 (탐색 버튼)
- LoadingBar 컴포넌트 (로딩 상태 표시)

### 예상 작업량
- **F-01**: 1일 (프로젝트 세팅, 라우팅, 스타일)
- **F-02**: 3일 (webOS WebView API 조사, 통합, 이벤트 핸들링)
- **F-03**: 2일 (URLBar, 가상 키보드, 자동완성)
- **F-04**: 1일 (NavigationBar, 히스토리 스택)
- **F-05**: 0.5일 (LoadingBar, 진행률 표시)
- **총 예상**: 7.5일

### 병렬 실행 가능 여부
- F-01 완료 후 F-02 단독 실행 (의존성 체인 시작점)
- F-02 완료 후 PG-1 (F-03, F-04, F-05) 병렬 실행 가능 (충돌 없음)

### 기술적 리스크
- **High**: webOS WebView API 제약 (iframe vs webOSBrowser API)
  - 대응: webOS 공식 문서 조사, 샘플 앱 참조
- **Medium**: 리모컨 가상 키보드 UX (입력 속도 느림)
  - 대응: 자동완성, 검색 엔진 연동으로 URL 입력 최소화

## Milestone 2: Essential Features — 탭, 북마크, 히스토리

### 목표
실사용에 필요한 탭 관리, 북마크, 히스토리 기능을 추가하여 일상적인 웹 브라우징을 지원

### 포함 기능
- **F-06**: 탭 관리 시스템
- **F-07**: 북마크 관리
- **F-08**: 히스토리 관리
- **F-09**: 검색 엔진 통합
- **F-10**: 에러 처리

### 의존성
- M1 완료 필요 (F-01, F-02, F-03 완료)

### 완료 기준
- 다중 탭 생성/전환/닫기 동작
- 현재 페이지 북마크 추가/편집/삭제
- 북마크 폴더 구조 관리
- 방문 기록 자동 저장 및 조회
- URLBar에서 검색어 입력 시 검색 엔진 사용
- 연결 실패, 404 등 에러 발생 시 에러 페이지 표시
- Netflix, Naver 등 주요 사이트 탭 전환하며 정상 동작

### 주요 산출물
- TabManager 서비스 (탭 상태 관리)
- TabBar 컴포넌트 (탭 리스트 UI)
- BookmarkManager 서비스 (IndexedDB 저장)
- BookmarkPanel 컴포넌트 (북마크 리스트 UI)
- HistoryManager 서비스 (IndexedDB 저장)
- HistoryPanel 컴포넌트 (히스토리 리스트 UI)
- SearchEngine 서비스 (검색 쿼리 생성)
- ErrorPage 컴포넌트 (에러 안내 화면)

### 예상 작업량
- **F-06**: 3일 (TabManager, TabBar, 메모리 최적화)
- **F-07**: 2일 (BookmarkManager, BookmarkPanel, CRUD)
- **F-08**: 2일 (HistoryManager, HistoryPanel, 날짜별 리스트)
- **F-09**: 1일 (SearchEngine, URLBar 연동)
- **F-10**: 1일 (ErrorPage, 에러 이벤트 핸들링)
- **총 예상**: 9일

### 병렬 실행 가능 여부
- F-06 단독 실행 (WebView와 밀접, 충돌 위험)
- F-06 완료 후 PG-2 (F-07, F-08, F-09) 병렬 실행 가능
  - 주의: F-09가 URLBar 수정 필요 (F-03 완료 후 시작)
- F-10 단독 실행 (WebView 에러 핸들링, 충돌 위험)

### 기술적 리스크
- **High**: 탭별 웹뷰 인스턴스 관리 (메모리 제약)
  - 대응: 비활성 탭 웹뷰 일시정지, 탭 수 제한 (최대 5개)
- **Medium**: IndexedDB 성능 (북마크/히스토리 대량 데이터)
  - 대응: 인덱스 최적화, 페이징 처리

## Milestone 3: Advanced Features — 다운로드, 설정, 단축키

### 목표
다운로드 관리, 설정 화면, 리모컨 단축키, 보안 표시, 즐겨찾기 홈 화면 등 고급 기능 추가로 완성도 높은 브라우저 제공

### 포함 기능
- **F-11**: 설정 화면
- **F-12**: 다운로드 관리
- **F-13**: 리모컨 단축키
- **F-14**: HTTPS 보안 표시
- **F-15**: 즐겨찾기 홈 화면

### 의존성
- M2 완료 필요 (F-06, F-07, F-09 완료)

### 완료 기준
- 설정 화면에서 검색 엔진, 홈페이지, 테마 변경
- 브라우징 데이터(쿠키, 캐시, 히스토리) 삭제 가능
- 파일 다운로드 시작, 진행률 표시, 완료 알림
- 리모컨 채널/컬러 버튼으로 빠른 기능 실행
- HTTPS 사이트에 자물쇠 아이콘 표시
- HTTP 사이트 접속 시 보안 경고
- 홈 버튼 클릭 시 즐겨찾기 그리드 화면 표시
- 1시간 연속 사용, 10개 탭 열어도 안정적 동작

### 주요 산출물
- SettingsPanel 컴포넌트 (설정 항목 UI)
- Settings 서비스 (설정 저장/로드)
- DownloadManager 서비스 (webOS 다운로드 API)
- DownloadPanel 컴포넌트 (다운로드 리스트 UI)
- KeyboardHandler 서비스 (리모컨 단축키 매핑)
- SecurityIndicator 컴포넌트 (보안 상태 아이콘)
- HomePage 컴포넌트 (즐겨찾기 그리드)

### 예상 작업량
- **F-11**: 2일 (SettingsPanel, 설정 항목, 데이터 삭제)
- **F-12**: 2일 (DownloadManager, DownloadPanel, webOS API)
- **F-13**: 1일 (KeyboardHandler, 단축키 매핑)
- **F-14**: 1일 (SecurityIndicator, 프로토콜 체크, 경고)
- **F-15**: 1.5일 (HomePage, 북마크 그리드 뷰)
- **총 예상**: 7.5일

### 병렬 실행 가능 여부
- F-11 단독 실행 (F-09 의존성, 설정 전역 영향)
- F-11 완료 후 PG-3 (F-12, F-13, F-14) 병렬 실행 가능
  - 주의: F-14가 URLBar 수정 필요 (F-09 완료 후 시작)
- F-15 단독 실행 (F-07 의존성, HomePage 라우팅)

### 기술적 리스크
- **High**: webOS 다운로드 API 제약 (파일 저장 경로, 권한)
  - 대응: webOS 공식 문서 조사, 대체 방안 (외부 링크 열기)
- **Low**: 리모컨 단축키 충돌 (webOS 시스템 키)
  - 대응: webOS 예약 키 회피, 커스터마이징 가능 설정
